# ContentCapture Pro - Quick Start Guide

Get up and running in 5 minutes!

---

## Step 1: Install AutoHotkey v2

1. Go to [autohotkey.com](https://www.autohotkey.com)
2. Click **Download**
3. Choose **AutoHotkey v2.0** (not v1.1)
4. Run the installer with default settings

---

## Step 2: Download ContentCapture Pro

1. Download the ZIP from this repository
2. Extract ALL files to a folder (e.g., `C:\ContentCapture-Pro`)
3. Keep all files together - they need each other!

---

## Step 3: Run the Script

1. Double-click `ContentCapture.ahk`
2. You'll see a small icon appear in your system tray
3. Right-click the tray icon to access the menu

---

## Step 4: Make Your First Capture

1. Open any webpage in your browser
2. (Optional) Highlight some text you want to save
3. Press `Ctrl+Alt+G`
4. Enter a short name like `test1`
5. Fill in any additional fields (or leave blank)
6. Click **Save**

**Congratulations! You made your first capture!**

---

## Step 5: Use Your Capture

Now the magic happens:

1. Open any application (Word, Notepad, email, chat, anything)
2. Type: `test1` then press **Space** or **Enter**
3. Watch your captured content appear instantly!

---

## Essential Shortcuts

| What You Want | What You Press |
|---------------|----------------|
| Capture a webpage | `Ctrl+Alt+G` |
| Browse all captures | `Ctrl+Alt+B` |
| Quick search | `Ctrl+Alt+Space` |
| New manual capture | `Ctrl+Alt+N` |

---

## Common Suffixes

If your capture is named `test1`:

| Type This | What Happens |
|-----------|--------------|
| `test1` | Paste the full content |
| `test1go` | Open the original URL |
| `test1em` | Create an email with the content |
| `test1fb` | Share to Facebook |
| `test1x` | Share to Twitter/X |
| `test1vi` | Edit the capture |

---

## Tips for Success

### Good Capture Names
- Keep them short: `recipe1`, `news23`, `howto5`
- Make them memorable: `chickensoup`, `pythontip`, `taxinfo`
- Avoid spaces and special characters

### Organize with Tags
- Add tags when capturing: `food`, `work`, `reference`
- Filter by tags in the Capture Browser

### Use Favorites
- Star your most-used captures
- Access favorites from the tray menu
